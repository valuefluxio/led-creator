Overview

What is it?

LED screens have a pixel pitch — the spacing between LEDs — that combined with physical size determines the exact screen resolution.

This tool calculates that resolution and renders a canvas at exact pixel dimensions — so what you design is what the screen shows.

Browser-based No install Single HTML file
02 / 09

The Math

Pixel Resolution Formula

pixelsWide = round( widthMeters × 1000 / pitchMm ) pixelsTall = round( heightMeters × 1000 / pitchMm )
// Example: 4m × 2m screen at P4 pixelsWide = round( 4000 / 4 ) = 1000 pixelsTall = round( 2000 / 4 ) = 500 // Result: 1000 × 500 px canvas

Quick Start

Getting Started

  • 01 Open the app — a setup modal appears automatically
  • 02 Enter screen width & height in meters
  • 03 Select LED pitch (P1.5 → P20), or type a custom mm value
  • 04 Live preview shows exact resolution and total pixel count
  • 05 Click "Create Content →" to open the editor

Tip: the ✎ Edit button in the top bar reopens the modal at any time.

Content

5 Animation Modes

Color Test

Broadcast test pattern — color bars, grayscale ramp, crosshair, corner markers. Breathing & cycle.

Scroll Text

Text scrolls horizontally. Customise text content, color, background, and font size.

Color Seq

Define a list of colors with per-step durations and optional fade transitions.

Patterns

Pulse · Wave · Sweep · Checker · Sparkle. Two-color configurable animated geometrics.

Images

Upload one or more images, set display duration per frame. Loops automatically.

05 / 09

Export

Export to Video

  • 01 Click ⬇ Export in the toolbar
  • 02 Set loop duration — 2s / 5s / 10s / 20s
  • 03 Drag the in / out handles on the timeline to select your range
  • 04 Click ⏺ Record — animation auto-seeks to in-point and records at 30fps
  • 05 File downloads automatically (MP4 where supported, WebM otherwise)

Deployment

On a Real LED Screen

Option A

Direct Connection

Connect the LED screen as a second monitor → open the app in browser → click ⛶ Full → drag the window to the LED display. No export needed.

Option B

Video File

Export a video and load it into the screen's media player — NovaStudio, Verypixel, etc. WebM → MP4 via HandBrake if needed.
